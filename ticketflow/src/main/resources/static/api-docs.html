🎫 TicketFlow API 文档

高并发票务系统 · 所有接口均以 /api 为前缀 · 除标注"公开"外均需 Bearer Token

Base URL: http://localhost:8080

📦 用户模块 — /api/user

POST /api/user/login 用户登录公开

Request Body (JSON)

{"username": "admin", "password": "admin123"}

Response

{"code": 200, "data": {"token": "eyJ...", "nickname": "管理员"}, "message": "操作成功"}
POST /api/user/register 用户注册公开

Request Body (JSON)

{"username": "test", "password": "123456", "nickname": "测试", "email": "test@t.com"}
GET /api/user/info 获取当前用户信息认证

Headers

Authorization: Bearer {token}
PUT /api/user/password 修改密码认证

Request Body

{"oldPassword": "admin123", "newPassword": "new123456"}
GET /api/user/list 用户列表（管理员）认证

Query Params

?current=1&size=10&keyword=admin
PUT /api/user/status/{userId} 启用/禁用用户认证

Request Body

{"status": 1}

📅 活动模块 — /api/event

GET /api/event/list 活动列表公开

Query Params

?current=1&size=12&keyword=周杰伦
GET /api/event/{id} 活动详情公开
POST /api/event 创建活动（管理员）认证

Request Body

{"eventName":"周杰伦演唱会","venue":"鸟巢","startTime":"2026-07-15 19:00:00", "endTime":"2026-07-15 22:00:00","onSaleTime":"2026-07-01 10:00:00", "totalSeats":5000,"merchantId":1,"description":"..."}
PUT /api/event/{id} 编辑活动认证
PUT /api/event/status/{id} 修改活动状态认证

Request Body

{"status": 1} (0未开售 1热卖中 2已售罄 3已结束)
DELETE /api/event/{id} 删除活动认证

🛒 订单模块 — /api/order

POST /api/order/seckill/{eventId} 秒杀抢票认证

Query Params

?seatIds=1001,1002,1003 (1-5个座位ID，逗号分隔)

Response (成功)

{"code":200,"data":{"success":true,"orderNo":"202606132035301234567890","message":"3张票出票处理中"}}

Response (库存不足)

{"code":409,"message":"库存不足，剩余票数不够"}

Response (重复购买)

{"code":409,"message":"您已抢过该活动的票，每人限购1单"}
GET /api/order/list 我的订单列表认证

Query Params

?current=1&size=10
GET /api/order/{id} 订单详情认证
PUT /api/order/cancel/{id} 取消/退款订单认证

业务规则

待支付(0) → 取消 → 状态变2(已取消)，释放座位 已支付(1) → 退款 → 状态变3(已退款)，释放座位，恢复Redis库存 已取消(2) → 报错 5003 已退款(3) → 报错 5004

💺 座位模块 — /api/seat

GET /api/seat/list/{eventId} 获取活动座位公开
POST /api/seat/batch 批量创建座位（管理员）认证

Request Body

{"eventId":1,"sectionName":"A区","rowCount":5,"seatsPerRow":10,"price":380}
